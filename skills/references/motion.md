# 动效 (Motion - 自定义)

## 核心摘要 (TL;DR)

1. **对于 life-ds 组件：请使用组件内置的动效。** 不要使用自定义弹簧参数覆盖组件的默认行为。
2. **本页预设仅适用于非组件的自定义元素**（例如自定义对话框/菜单、小型装饰性掉落物）。
3. 先确定 **交互目标**，然后选择对应的 **速度预设**（快速/慢速/轻微回弹）。确保相似场景下的动效保持一致。

## 核心动效原则 (Motion Principles)

- **感知确认**：动效应符合真实世界的物理特性和直觉反馈。避免出现“我点击了但没反应”或“我不理解发生了什么变化”的情况。
- **及时反馈**：触发时机、持续时间和曲线应匹配用户的关注点；不要让关键反馈显得迟钝。
- **无缝衔接**：平滑的动效不仅仅是“越快越好”；应匹配连续手势和过渡时机，避免突兀的跳变。
- **层级清晰**：利用动效来强化结构和空间关系（进入/退出、前景/背景、归属关系），而不是混淆层级。
- **目标明确**：每一个动效都应该回答“为什么它是必要的”；过大的幅度或过多的元素会分散并抢夺用户的注意力。
- **一致性**：在相似场景下保持动效模式、速度曲线和反馈的一致性，以建立可预测性。
- **情感化**：动效应与功能/产品想要传达的情感保持一致；避免“只为了设计师而存在”的动效。

## 动效预设 (仅适用于非组件的自定义元素)

使用 `type: 'spring'` 配合 `stiffness` (劲度) / `damping` (阻尼)。下方的数值是推荐的预设值——请在相同场景下复用它们。

### 速度预设 (通用)

- **慢速 (\~350ms)**: `stiffness: 322.27, damping: 32.31`
- **快速 (\~200ms)**: `stiffness: 947.29, damping: 61.56`
- **弹出/Pop-up (\~300ms, 轻微回弹)**: `stiffness: 438.65, damping: 29.32`

### 场景映射 (推荐)

- **自定义对话框 (非组件)**: `stiffness: 631.65, damping: 50.27`
- **分段控件焦点滑动** (\~300ms, 0 回弹): `stiffness: 438.65, damping: 37.70`
- **自定义菜单** (\~300ms, 0 回弹): `stiffness: 438.65, damping: 37.70`
- **小型装饰性掉落元素** (\~300ms, 0 回弹): `stiffness: 438.65, damping: 37.70`
- **底页 (自定义 Sheet 滑动)**: `stiffness: [986.96, 438.65], damping: [62.83, 41.89]`

### 错误抖动 (非弹簧)

- `bezierEase: [0.42, 0, 0.58, 1]`
- `translationX: [0, 8, -8, 5, -5, 2, -2, 0]`

### 示例 (Framer Motion)

```ts
transition={{
  type: 'spring',
  stiffness: 438.65,
  damping: 37.7,
}}
```

## Web 辅助工具 (推荐)

搜索代码库，如果存在请优先使用现有的辅助函数：

- `useSpringMotion()` — “传统弹簧动效”。
- `usePopMotion()` — 弹出/轻微回弹；通常更具可预测性。
