# 常用脚本

本目录用于沉淀 `life-design-system-pc` 相关的常用脚本与维护命令。

当前已整理的脚本文档：

- [update-tokens.md](update-tokens.md)：更新设计 token、文本样式和效果样式
- [update-skills.md](update-skills.md)：同步 skill 文档产物
- [update-components.md](update-components.md)：更新组件库产物

建议优先阅读更新类脚本：

1. token 更新：`sync-tokens`
2. skill 文档同步：`sync-skills`
3. 组件产物更新：`build --workspace=@life-ds/components-web`
